ESTRUCTURA DE CLASES - Roles_Principal.pyw
════════════════════════════════════════════════════════════════════════════

CLASE 1: ObtenerDatos
├── __init__(server, database, username, password)
│   └── Configura conexión a SQL Server
│
├── _get_connection()
│   └── Retorna conexión ODBC a SQL Server
│
└── obtener_datos_empleado_rapido(periodo, cedula_o_nombre)
    └── Retorna: pandas.Series con datos de UN empleado
        Flujo:
        1. Busca empleado por EMPLEADO, CEDULA o nombre
        2. Obtiene movimientos (RPINGDES → RPHISTOR)
        3. Consolida conceptos por mapeo
        4. Calcula ingresos, egresos, neto
        5. Obtiene nombres descriptivos de DBTABLAS


CLASE 2: GeneradorPDFs
├── __init__()
│   └── Configura parámetros de conexión BD
│
├── obtener_datos_bd(periodo)
│   └── Retorna: pandas.DataFrame con TODOS los empleados del período
│       Flujo:
│       1. Obtiene empleados ACTIVOS (RPEMPLEA)
│       2. Obtiene movimientos consolidados
│       3. Hace JOIN empleados + movimientos
│       4. Traduce códigos a nombres descriptivos
│       5. Aplica lógica ANTIQUINC para FONDO_RESERVA
│
├── crear_pdf_empleado_bd(filename, row, start_date, end_date, logo_path)
│   └── Genera PDF simple (1 rol por página)
│
├── crear_pdf_empleado_doble(filename, row, start_date, end_date, logo_path)
│   └── Genera PDF con 2 roles idénticos por página
│       (para ahorro de papel, con línea divisoria)
│
├── dibujar_rol_en_posicion(c, row, start_date, end_date, width, height, y_offset, logo_path)
│   └── NÚCLEO DE GENERACIÓN PDF
│       Dibuja:
│       - Encabezado (título, empleado, período, depto, cargo)
│       - Tabla de ingresos/descuentos
│       - Totales
│       - Firma
│       - Logo (si aplica, en blanco y negro)
│
├── _format_cedula(cedula)
│   └── Convierte cédula float a string (ej: 920116811.0 → "920116811")
│
├── _safe_get_bd(row, column_name, default=0)
│   └── Obtiene valor de forma segura (null-safe)
│
├── _calculate_overtime_bd(row)
│   └── Suma SOBRETIEMPO_25 + 50 + 100
│
└── _calculate_reserve_fund_base_bd(row)
    └── Calcula base para FONDO_RESERVA


CLASE 3: RolesPrincipal (PRINCIPAL)
├── __init__(root)
│   ├── Configura ventana principal
│   ├── Crea instancias compartidas:
│   │   ├── self.obtener_datos = ObtenerDatos()
│   │   └── self.generador_pdf = GeneradorPDFs()
│   ├── Crea header
│   └── Crea Notebook con 2 pestañas
│
├── _crear_header()
│   └── Frame superior con título y subtítulo
│
├── _crear_visualizador()
│   └── Pestaña 1: Búsqueda individual + preview PDF
│       Widgets:
│       - vis_periodo_var (StringVar)
│       - vis_buscar_var (StringVar)
│       - vis_status (Label)
│       - vis_canvas (Canvas para PDF)
│       - vis_photo_image (PhotoImage para mostrar)
│       - vis_datos_actual (pandas.Series)
│
├── ═══ MÉTODOS VISUALIZADOR ═══
├── _vis_buscar()
│   └── Valida entrada y lanza thread de búsqueda
│
├── _vis_buscar_thread(periodo, filtro)
│   └── Ejecuta ObtenerDatos.obtener_datos_empleado_rapido()
│       Usa self.root.after() para actualizar UI
│
├── _vis_mostrar()
│   └── Genera PDF con GeneradorPDFs.crear_pdf_empleado_bd()
│       Calcula período correcto (YYYY-MM-01 a YYYY-MM-último_día)
│       Muestra en canvas con _vis_mostrar_pdf_como_imagen()
│
├── _vis_mostrar_pdf_como_imagen(ruta_pdf)
│   └── Convierte PDF a imagen con PyMuPDF
│       Muestra en canvas con zoom 1.5x
│       Permite scroll vertical
│
├── _vis_descargar()
│   └── Guarda PDF a carpeta seleccionada
│       Nombre: {cedula}_{nombre}.pdf
│
├── ═══ MÉTODOS GENERADOR ═══
├── _crear_generador()
│   └── Pestaña 2: Batch + configuración
│       Widgets:
│       - gen_periodo_var (StringVar)
│       - gen_filtro_var (StringVar)
│       - gen_formato_nombre (StringVar)
│       - gen_dos_por_hoja (BooleanVar)
│       - gen_incluir_logo (BooleanVar)
│       - gen_cedulas_text (Text)
│       - gen_barra_progreso (Progressbar)
│       - gen_etiqueta_estado (Label)
│       - gen_etiqueta_carpeta (Label)
│
├── _gen_seleccionar_periodo()
│   └── Diálogo modal para seleccionar AÑO + MES
│
├── _gen_seleccionar_carpeta()
│   └── Diálogo de carpeta para destino
│
├── _gen_toggle_logo()
│   └── Habilita/deshabilita botón "Seleccionar logo"
│
├── _gen_seleccionar_logo()
│   └── Diálogo de archivo para logo
│
├── _gen_iniciar()
│   └── Valida parámetros y lanza thread
│
├── _gen_ejecutar()
│   └── FLUJO PRINCIPAL DE GENERACIÓN:
│       1. Crea carpeta YYYY-MM en destino
│       2. Obtiene datos de BD (ObtenerDatos.obtener_datos_bd)
│       3. Aplica filtro de cédulas (si existen)
│       4. Aplica filtro de texto (si no hay cédulas)
│       5. Itera sobre empleados:
│          - Calcula progreso
│          - Genera nombre de archivo
│          - Crea PDF (simple o doble)
│       6. Muestra mensaje de éxito
│       7. Reactiva botón "Generar Roles"
│
└── _gen_nombre_archivo(row, output_folder, period_str)
    └── Genera nombre según formato seleccionado:
        ✓ cedula-nombre (defecto)
        ✓ nombre-cedula
        ✓ cedula-nombre-cargo
        ✓ cedula-nombre-depto
        ✓ nombre-cargo-cedula
        ✓ depto-nombre-cedula


FLUJO DE DATOS VISUALIZADOR
────────────────────────────
Usuario escribe cedula/nombre y período
         │
         ▼
_vis_buscar() → Validación
         │
         ▼ (threading.Thread)
_vis_buscar_thread() → ObtenerDatos.obtener_datos_empleado_rapido()
         │
         ▼ (self.root.after)
_vis_mostrar() → GeneradorPDFs.crear_pdf_empleado_bd() → tempfile
         │
         ▼
_vis_mostrar_pdf_como_imagen() → fitz.open() → PIL.Image → ImageTk.PhotoImage
         │
         ▼
Canvas.create_image() → Muestra en ventana


FLUJO DE DATOS GENERADOR
────────────────────────
Usuario selecciona período, carpeta, opciones
         │
         ▼
_gen_iniciar() → Validación
         │
         ▼ (threading.Thread)
_gen_ejecutar():
   1. GeneradorPDFs.obtener_datos_bd() → DataFrame TODOS
   2. Filtra cédulas específicas (si existen)
   3. O filtra por texto
   4. Para cada empleado en DataFrame:
      - _gen_nombre_archivo() → nombre
      - GeneradorPDFs.crear_pdf_empleado_bd() o create_pdf_employee_doble()
   5. Actualiza Progressbar cada iteración
   6. messagebox.showinfo() al finalizar

MANEJO DE THREADING
───────────────────
Visualizador:
  - _vis_buscar() lanza thread
  - _vis_buscar_thread() ejecuta en background
  - Usa self.root.after(0, función) para actualizar UI desde thread

Generador:
  - _gen_iniciar() lanza thread
  - _gen_ejecutar() ejecuta en background
  - Usa self.root.update_idletasks() para permitir actualizaciones
